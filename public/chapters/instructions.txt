Инструкция к мультимедийному учебному пособию
"История лингвистических исследований".


1.Необходжимые требования для работы с учебным пособием:

а) В учебном пособии были использованы специальные фонетические шрифты. 
Для работы требуется установить эти шрифты на компьютере пользователя. 
Если Вы работаете с CD-версией, то необходимый набор шрифтов содержится 
в папке fonts-manual в корневом каталоге учебного пособия. 
Для установки шрифтов нужно нажать:
Пуск-> Панель управления ->Шрифты.
Затем: Файл -> Установить шрифт.
Указать местоположение шрифтов на диске (папка fonts-manual). 
Затем шрифты установить. 

б) Учебное пособие лучше всего отображается в браузере Internet Explorer. 
Тестовые задания, включающие javascript, могут не работать в других браузерах. 

в) Для просмотра некоторых приложений в формате PDF на Вашем компьютере 
должна быть установлена программа Acrobat Reader, которая будет автоматически 
вызываться при запуске приложения. 

2. В каталоге учебного пособия запускается файл INDEX.HTML

3. Интерфейс учебника представляет собой фреймовую структуру и содержит
несколько основных окон:
а) вернее окно с логотипом,
б) левой окно с содержанием учебного пособия,
в) нижнее окно с ссылками на первую  страницу учебного пособия, список авторов,
требования, обозначения.
г) центральное окно учебного пособия, в котором автоматически
отображаются страницы основного
текста, рисунки, таблицы.
Если автоматически они не отображаются в центральном окне, то необходимо в браузере 
отключить функцию отображения ссылок в новом окне.

4.Содержание учебного пособия включает:

а) ссылки на основное текст учебного пособия,
б) ссылки на список приложений
в) список литературы
г) список контрольных упражнений, представляющих собой автоматизированные тесты для самоконтроля
д) именной указатель
е) терминологический словарь, включающий список основных терминов

5. Все иллюстрации и таблицы находятся под ссылками в соответствующих разделах учебного курса.
